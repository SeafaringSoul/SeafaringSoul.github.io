Rust 环境搭建安装

通过 rustup下载Rust，这是一个管理 Rust 版本和相关工具的命令行工具。下载需要联网

rustup 命令

  • update : 升级rust工具链和rustup本身
  • self uninstall : 卸载rust
  • doc : 在浏览器中打开离线文档

VsCode 配置

推荐插件

  • rust-analyzer
  • even-better-toml 支持 .tom 文件完整性
  • crates

cargo 学习

cargo 项目生成

发布构建

cargo命令总结

cargo 国内镜像更改

  • .cargo目录下创建文件config
  • 放到 $HOME/.cargo/config 文件中

总结

  1. 安装与命令：通过特定命令联网安装 Rust，rustup 工具提供了版本管理等功能，其常用命令有 update、self uninstall、doc 等。
  2. 开发环境配置：VsCode 中推荐安装 rust-analyzer 等插件辅助开发。
  3. 项目操作与构建：cargo 用于项目管理，可创建、构建、运行项目，构建分开发与面向用户两种模式，还可进行错误检查，同时可更改国内镜像优化下载。
