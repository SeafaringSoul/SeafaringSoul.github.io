变量和可变性

变量

  1. 声明方式let
  2. 变量默认是不可改变的（immutable）。
  3. 当变量不可变时，一旦值被绑定一个名称上，你就不能改变这个值。
  4. 尽管变量默认是不可变的，但是可以在变量名前添加 mut 来使其可变。

常量

  1. 声明方式const
  2. 不允许对常量使用 mut。常量不光默认不可变，它总是不可变
  3. 常量可以在任何作用域中声明，包括全局作用域
  4. 常量只能被设置为常量表达式，而不可以是其他任何只能在运行时计算出的值
  5. Rust 对常量的命名约定是在单词之间使用全大写加下划线

隐藏

  1. 两个相同的变量名称，第二个变量覆盖了第一个变量。任何使用该变量都是在使用第二个变量，直到有第三个变量覆盖，或者作用于结束。
  2. 可以用相同变量名称来隐藏一个变量，以及重复使用 let 关键字来多次隐藏

数据类型

  • 在 Rust 中，每一个值都属于某一个 数据类型（data type），这告诉 Rust 它被指定为何种数据，以便明确数据处理方式。我们将看到两类数据类型子集：标量（scalar）和复合（compound）。

标量类型（scalar）

  • 标量（scalar）类型代表一个单独的值。
  • ust 有四种基本的标量类型:
    • 整型 int
    • 浮点型 float
    • 布尔型 bool
    • 字符型 string

整型

  • 整数 是一个没有小数部分的数字
  • u 代表无符号，i 代表有符号
  • 固定大小的整数类型：
    • 有符号整数类型：i8, i16, i32, i64, i128
      • 这些类型用于表示有符号的整数（可以是正数或负数）。
      • i32 是常用的 32 位整数类型，i64 是常用的 64 位整数类型。
    • 无符号整数类型：u8, u16, u32, u64, u128
      • 这些类型用于表示无符号的整数（只能是正数或零）。
  • 平台相关的整数类型：
    • isize：根据目标系统架构（32 位或 64 位）变化的有符号整数类型。
      • 32 位系统上为 32 位，64 位系统上为 64 位。
    • usize：根据目标系统架构（32 位或 64 位）变化的无符号整数类型。
      • 32 位系统上为 32 位，64 位系统上为 64 位。
      • 通常用于表示索引或与内存相关的大小。

总结

  1. Rust 没有 int 和 uint 类型的概念。
  2. isize 和 usize：这两个类型用于处理与系统架构相关的整数，通常用于内存地址、数组索引等。
  3. Rust 的类型系统更加明确和精细，避免了 int 和 uint 这类不明确的类型。
  4. 通过使用 i32, u64, isize, usize 等类型，Rust 可以为不同用途的整数提供明确的类型保证。

浮点型（Floating-point Numbers）

Rust 提供了两种原生的浮点数类型：f32 和 f64。它们用于表示带小数点的数字，分别占用 32 位和 64 位内存空间。

  • f32：占用 32 位内存，单精度浮点数。
  • f64：占用 64 位内存，双精度浮点数。

默认类型为 f64，因为它在现代 CPU 上的性能与 f32 相似，但提供更高的精度。

所有浮点数类型在 Rust 中都是有符号的，意味着它们可以表示正数、负数和零。

布尔型（Boolean）

在 Rust 中，布尔类型用 bool 表示，具有两个可能的值：true 和 false。布尔值主要用于条件表达式，如 if 语句。

布尔值的主要用途是控制流操作，特别是在 if 表达式中。

字符类型（char）

在 Rust 中，char 类型用于表示单个字符。它与其他语言中的字符类型（如 C 中的 char）不同，Rust 的 char 是一个 4 字节（32 位）的 Unicode 标量值。由于它是 Unicode 标量值，所以 char 类型能够表示更多字符，包括但不限于 ASCII 字符。

  • 声明方式：char 字面量使用单引号 ‘ 来声明，而字符串使用双引号 “。

  • Unicode 支持：Rust 的 char 类型可以表示广泛的字符，包括带变音符号的字母、中文、日文、韩文字符以及表情符号（emoji）。char 类型的值范围从 U+0000 到 U+D7FF 和 U+E000 到 U+10FFFF。

  • 与字符的直觉理解不同：虽然我们通常把一个字符理解为一个字母或符号，但在 Unicode 中，“字符”并不等同于 char 类型。例如，一个 char 可以是一个表情符号或者其他特殊符号。

复合类型

复合类型允许将多个值组合成一个类型。在 Rust 中，有两种主要的复合类型：元组（tuple） 和 数组（array）。

元组是将多个值组合成一个复合类型的方式，可以包含不同类型的元素。元组的长度是固定的，定义后不能增大或缩小。

数组类型

数组是包含多个相同类型元素的集合，长度固定。与元组不同，数组中的元素类型必须相同。Rust 中的数组在栈上分配内存。

函数

函数的定义和调用

  • 定义函数：使用 fn 关键字，函数名通常使用蛇形命名法（snake case）。
  • 调用函数：通过函数名加括号（带或不带参数）来调用。

函数的参数

  • 参数在函数签名中声明，必须指定类型。
  • 语法：参数名: 参数类型。
  • 参数 x 的类型为 i32，调用时需传入一个整数。

多个参数

  • 函数可以有多个参数，参数之间用逗号分隔。

函数的返回值

  • 函数可以返回值，通过 -> 指定返回值的类型。
  • 返回值是函数体中最后一个表达式的值，且不能带分号。
  • 可以显式使用 return 提前返回。

注意点

  • 如果在返回值后加了分号，Rust 会将其视为语句而非表达式，从而报错。

语句与表达式

  • 语句（Statements）：执行操作，但不返回值。例如 let x = 6;。
  • 表达式（Expressions）：产生值，可以嵌套在其他表达式中或赋值给变量。例如 5 + 1。
  • 花括号中的块是一个表达式，它计算 x + 1 的值并赋给 y。

总结

  • 函数是 Rust 的核心构建块，用于组织和复用代码。
  • 函数参数必须指定类型，返回值通过表达式确定。
  • 区分语句和表达式对理解 Rust 的函数行为至关重要。

控制流（Control Flow）

控制流用于决定代码的执行路径，包括条件语句和循环。

条件语句：if

基本语法

  • 使用 if 语句根据条件执行不同代码块。
  • 条件必须是布尔值，Rust 不会自动将其他类型转换为布尔值。

循环（Loops）

loop 循环

  • 无限循环，使用 break 退出。
  • 可以为循环标注标签以区分嵌套循环。

带返回值的 loop

  • break 可以返回一个值。

while 循环

  • 根据条件运行代码块，条件为假时停止。

for 循环

  • 遍历集合或范围。
  • 更安全且简洁。

使用 range 遍历

  • for 循环配合范围表达式遍历数字序列。

总结

  • if 和 else 用于条件分支，必须返回布尔值。
  • loop 是无限循环，可以通过 break 中断。
  • while 循环根据条件运行代码。
  • for 循环是遍历集合或范围的首选方式。
